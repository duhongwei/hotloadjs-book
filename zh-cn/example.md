# 演练

为了方便，演练放在了hotloadjs项目里

## 环境
- linux
- nodejs > 6.0 最好用最新版本
- npm >3.0
- git >1.7

## 准备步骤

1. clone 项目
``` shell
git clone https://github.com/duhongwei/hotloadjs.git
```
2. 安装
默认会建一个hotloadjs目录 ，进入hotloadjs目录
``` shell
npm install
```
3. 执行初始化

安装完成后，执行

``` shell
npm run example
```
4. 访问页面

http://localhost:3000/examples/page

> 注意：如果你是通过远程访问主机，需要把localhost换成 ip 地址


每个演练都是独立的，但都需要执行上面的步骤。
